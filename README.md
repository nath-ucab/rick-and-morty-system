# Rick and Morty System

Aplicación web para gestionar personajes y episodios de Rick and Morty.

## Características

### Autenticación
- Inicio de sesión
- Registro de usuarios
- Recuperación de contraseña

### Personajes
- Lista con ID, Nombre, Especie, Género y Tipo
- Búsqueda en tiempo real
- Ordenamiento por columnas
- Detalle con fotografía
- Edición de datos

### Episodios
- Lista con ID, Nombre, Fecha y Código
- Búsqueda en tiempo real
- Ordenamiento por columnas
- Detalle técnico
- Edición de datos

### Diseño
- Modo claro/oscuro
- Responsive (móvil, tablet, escritorio)
- Offline con Service Worker

## Tecnologías
- HTML5
- CSS3 (variables CSS)
- JavaScript (ES6+)
- Service Worker
- LocalStorage
- Rick and Morty API
